# Ecola

A lightweight Python radio simulator inspired by classic GTA-era radio systems.

## Features
- Continuous playback across station switches
- WAV-based radio archives
- Persistent playback state between runs

## Non-goals
- Music player replacement
- Metadata-heavy radio configs
- Bundled audio content

## Legal
This project does not include any copyrighted audio.
